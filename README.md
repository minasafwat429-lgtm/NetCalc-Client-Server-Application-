# NetCalc (Client-Server Application)

NetCalc is a Java-based calculator application built using a **Client-Server architecture**, capable of handling **multiple concurrent network requests over TCP sockets**.  
This project demonstrates core concepts in **network programming, thread lifecycle management, data streams, and robust error handling** with interactive GUIs.

⚠️ This project is intended for **educational purposes and learning network programming concepts** only.

---

## Features

- Handles **concurrent client requests** using multi-threading
- Interactive GUIs for both client and server using Java Swing
- Real-time server logging of client activity
- Robust error handling during execution
- Practical demonstration of **Client-Server architecture** and **TCP/IP communication**

---

## Technologies Used

- Java
- TCP/IP Sockets
- Multi-threading
- Java Swing
- Object-Oriented Programming (OOP)

---

## Requirements

- Java JDK 8 or higher
- IDE such as IntelliJ IDEA or Eclipse
- Basic understanding of network programming and TCP/IP
